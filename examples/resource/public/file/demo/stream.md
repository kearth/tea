## 🏯 皇家建筑与宫殿
1. **故宫（紫禁城）**
   - 明清两代的皇宫，现为故宫博物院。
   - 特色：世界上保存最完整的木质结构古建筑群。
   - 推荐游览路线：午门 → 太和殿 → 乾清宫 → 御花园。
2. **天坛**
   - 明清皇帝祭天祈谷的场所。
   - 特色：以祈年殿、回音壁、圜丘坛为代表。
   - 推荐体验：清晨看当地人晨练、太极拳。
3. **颐和园**
   - 清朝皇家园林，世界文化遗产。
   - 特色：昆明湖、万寿山、长廊、佛香阁。
