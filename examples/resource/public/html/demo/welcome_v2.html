Why Choose Tea?

Designed for developers who value clean code and rapid development.

Clean Architecture

Promotes a clear separation of concerns, making your codebase maintainable, testable, and scalable from day one.

Blazing Fast

Built on top of Gin, Tea leverages Go's performance to handle thousands of concurrent requests with minimal latency.

Extensible & Modular

Easily extend with middleware, custom handlers, and seamlessly integrate with your favorite Go libraries.

Developer First

Intuitive API design, comprehensive documentation, and a gentle learning curve for developers of all levels.

Secure by Default

Rich Tooling

Quick Start

Get your Tea application up and running in less than a minute.

1

Install Tea CLI

go install github.com/tea-framework/tea/cmd/tea@latest
2

Create a New Project

tea new my-awesome-app && cd my-awesome-app
3

Run the Development Server

tea run

That's it! Your server is now running at http://localhost:8080

Simple & Expressive Code

See how easy it is to define routes and handlers.

main.go 
package main

import (
    "github.com/gin-gonic/gin"
    "github.com/tea-framework/tea" // Tea's core package
)

func main() {
    // Initialize Tea with default middleware (logger, recovery)
    r := tea.Default()

    // Define a simple GET route
    r.GET("/", func(c *gin.Context) {
        c.JSON(200, gin.H{
            "message": "Welcome to Tea Framework!",
        })
    })

    // Define a route with a parameter
    r.GET("/user/:name", func(c *gin.Context) {
        name := c.Param("name")
        c.String(200, "Hello %s", name)
    })

    // Start the server on port 8080
    r.Run(":8080")
}
